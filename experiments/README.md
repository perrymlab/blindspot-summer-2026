# Experiments

Experiment folders contain protocols, commands, configs, and notes. Large generated outputs belong in `results/` or ignored local run folders.

- `week03-baseline/`: clean BoT-SORT-ReID baseline.
- `week04-poisoning/`: embedding-space poisoning experiments.
- `week06-detector/`: camera-level detector experiments.
- `week07-scalability/`: scenario and boundary-condition experiments.
